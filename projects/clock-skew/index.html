Measuring clock skews of servers

In this project, we are interested in measuring the clock skew, if any, of an HTTP server. To this end, we issue HEAD requests to different HTTP servers at a low frequency (i.e., a maximum of 6 times per day) and retrieve the Date HTTP header in the response. By comparing this timestamp against a reference clock, we can estimate the clock skew on the server that served this timestamp.
